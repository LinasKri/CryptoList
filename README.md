# Crypto single-page application

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Start node server

Find 'config.js' in 'my-server' folder and replace XXXXs with your API key.

Then in terminal (from project root folder):

```sh
cd my-server
node server.js
```

Refresh your localhost page (e.g. http://127.0.0.1:5173/) and Voil√† !!!

### Compile and Minify for Production

```sh
npm run build
```
